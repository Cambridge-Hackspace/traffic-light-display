# traffic-light-display

Software to drive the traffic light display at the
[Cambridge Hackspace](https://cambridgehackspace.com).

*You are on the `archive/ed` branch.*

This branch is no longer maintained. It contains the software originally
developed by Edward "Ed" Klacza (and potentially other contributors), which was
used to display a scrolling marquee with the text "Cambridge Hackspace ☺ ☺ ☺".
The software was deployed to an ESP32 DevKit v1 and is no longer used in
production.
